

# Sinter Compositions 
封装一些个人常用的 vue3 composition 代码 


## install 安装 

```bash
npm i sinter-compositions
// or 
yarn add sinter-compositions
```


## 使用

只需要进行个别 composition 的导出即可, 如下 

```ts
import { useNumberScope } from "sinter-compositions"
// ..... 这样你就可以使用了
```
